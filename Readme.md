### 系统简介（introduction）

1. 开发工具：

nodejs + vscode + MicroCommunity(HC小区后端)

2. 官方网站

   http://www.homecommunity.cn
   
   技术业务交流群QQ: 827669685

3. 环境搭建

    将app.js 中代码

    > app.use('/callComponent',proxy('http://192.168.0.104:8012',opts));
    中IP换为后端MicroCommunity项目FrontService服务地址
