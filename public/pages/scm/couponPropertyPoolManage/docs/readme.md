## 优惠券操作流程说明

优惠券是为了有效的提高物业缴费率而设计的，其目的是物业在收费月赠送业主优惠券（比如购物券，家政服务券，维修券和停车劵）的方式，</br>
促进业主能够按时及时的缴费，具体操作简单流程如下：</br>
1、物业在优惠券市场购买（这个是系统运营方和第三方平台集成的情况下才能使用，比如淘宝购物券，京东购物券等）</br>
或者可以在优惠券页面自己制作；</br>
2、设置赠送规则，在赠送规则页面添加赠送规则，并且设置要赠送的优惠券和设置缴什么费用时赠送;</br>
3、收费人员通过业务受理缴费页面缴费时或者业主在手机端自主缴费时就会赠送相应的优惠券;</br>
4、业主可以在手机端 我的 卡券页面查看 缴费获取到的优惠券;</br>
5、点击优惠券就可以出现核销二维码，可以通过 物业手机端优惠券核销功能核销，</br>
物业购买的优惠券可以通过 第三方平台核销即可，<br/>
停车券可以直接填写车牌号赠送停车券，赠送后停车券有效期为24小时;</br>

### 优惠券市场

平台对接了第三方平台情况下，第三方平台可以推自己的优惠券到物业系统，物业可以通过购买的方式获取优惠券，此功能必须系统运营方支持的情况下才能使用;</br>

### 优惠券

这里显示物业购买的优惠券和自己制作的优惠券，点击制作时 如果用途为停车券时，切记小时必须要输入正整数，不能输入汉子字母等</br>
数量为当前优惠券数量 应写一个比较大的数比如1000，否则如果优惠券用完时，就无法赠送<br/>
有效期：单位为天，应该要写天数 写入数字</br>
使用说明：请填写优惠券使用说明，方便业主自主能够操作</br>

### 赠送规则

添加赠送规则，填写合适的规则名称，选择赠送的优惠券，填写一次缴费赠送的优惠券数量</br>
赠送频率：
</br>
赠送一次： 缴费只赠送一次</br>
每月赠送一次：根据缴费时间段内包含的月份，每月赠送一次优惠券，比如：缴一年的物业费，每月赠送8张停车券</br>
添加缴费赠送，选择费用</br>
缴费开始时间-缴费结束时间：在这段时间内缴费即可享受该优惠券赠送</br>
缴费月:缴费多少个月赠送优惠券，缴费大于该缴费月只送一次，比如缴费月设置为12，那么缴费时如果缴费24个月 也只送一次，不会按倍数赠送</br>
